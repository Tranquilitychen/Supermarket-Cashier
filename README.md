# Supermarket-Cashier
超市收银系统

### 软件环境：Windows

### 开发工具：IDEA

### 项目部署: 
####         1. 下载项目到本地。
####         2. 导入项目到Eclipse 或者 IDEA,修改配置文件 db.properties中的数据库信息。
####         3. 运行项目。
   
### 使用到的框架：
####    前端： layui
####    后端： SpringMVC + Spring + MyBatis + Maven
####    数据库： MySQL
